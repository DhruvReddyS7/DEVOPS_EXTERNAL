# DevOps Lab Survival Kit 2025-26

A fast React + Vite + Tailwind CSS single-page exam copy tool for DevOps Lab preparation. It contains 27 set-wise programs, simple installation commands, search, filters, and copy buttons for commands/code.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Notes

- Downloads and non-exam dashboard features are intentionally removed.
- Every program, command, and full set has copy buttons for quick exam use.
- The app is a single-page website for fast navigation during exam preparation.
